Conclusion

In this project, we need to predict the popularity of online news from three different sources:

  • Facebook
  • GooglePlus
  • LinkedIn

This task can be defined as both text mining and classification. The data has large sampel size and small feature size.

  • We define a data model to predict the popularity of given news.
  • We report the performance of the data model in three experiments
  • The prediction is close to ground truth and the result is interpretable
  • This prediction model can be generally used in different applications. For example, before we advertise products online, we can predict, in comparison with existing advertisements, how popular the new advertisement will be. There are other applications, such as election campaign, news/posts recommendation, dynamic content generation and so on.
